# Identity Providers

Tiro.health supports multiple identity providers for user authentication:

## Supported Options

**Microsoft Entra ID (Azure AD)**
- Single Sign-On with Microsoft credentials
- Enterprise security features and compliance

**Google Identity**
- Authentication using Google accounts
- Support for Google Workspace organizations

**Auth0 Fallback**
- Email/password authentication with multi-factor authentication
- Available when external identity providers are not suitable

## Getting Started

Contact [support@tiro.health](mailto:support@tiro.health) to configure your preferred identity provider for your organization.
